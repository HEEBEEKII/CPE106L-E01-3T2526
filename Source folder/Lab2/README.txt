# Student Management System

A lightweight, CLI-based Python application designed to manage student records using fundamental data structures. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and efficient data handling in Python.

## Features

The system provides a simple interactive menu to perform the following:
* Create: Add new students with a unique ID and Name.
* Read: View a formatted list of all registered students and their grades.
* Update: Append new grades to a specific student's record.
* Delete: Remove a student record from the system using their ID.

## Technical Overview

This program utilizes a variety of Python's built-in data types to manage information efficiently:

- Dictionary: Used as the primary storage (students) for fast lookup via Student ID.
- Tuple: Stores the fixed record structure: (name, [grades]).
- List: A mutable collection inside the tuple to store and update student grades.
- String: Handles user input for names, IDs, and menu navigation.

## Installation & Usage

1. Prerequisites: Ensure you have Python 3.x installed on your machine.
2. Run: Open your terminal or command prompt and execute:
   python main.py

## How it Works

Upon launching, you will be presented with a menu to select an option (1-5). Note that data is stored in-memory; records will be cleared once the program is exited.
